## 프로젝트 소개
> AI 스피커와 Deep Learning 을 활용한 어르신 돌봄 시스템 입니다.어르신들의 안전을 위해 여러 정보들을 사회 복지사에게 제공하여 Deep Learning 을 통해 어르신의 행동, 가구의 열림과 닫힘 상태 등을 분석해서 평소와 다른 특이점이 발견되면 사회 복지사에게 알림을 보내 사고를 미연에 방지할 수 있도록 하였습니다. 
 

## ✔️ GUIDES

Xcode의 iOS 환경에서 실행하였습니다. Xcode, npm 설치가 되어있다고 가정합니다.

```shell
$git clone https://github.com/BugiCare/BugiCareServer_Flask.git](https://github.com/BugiCare/BugiCareAdminApp.git
$cd BugiCareAdminApp
$npm install
$cd ios
$pod install
$cd ..
$react-native run-ios 혹은 $npx react-native run-ios
```
